# DataCenter 整體前端架構及 UI 畫面及開發環境及流程 (1150325)

這份文件記錄了 DataCenter 前端的技術核心、設計美學以及 1150325 版本中最新的開發規範。

---

## 🎨 第一部分：UI 設計哲學 (Cyberpunk SCADA)

我們的設計語言核心是：**「專業、即時、光線感、戰情室視覺」**。

### 1. 核心配色
- **深空底色 (`#010613`)**：低疲勞感的深藍基調。
- **神經青色 (`#06b6d4`)**：主裝飾色，模擬雷射光束感。
- **警報紅色 (`#ef4444`)**：關鍵異常的高對比顯示。

### 2. 視覺特效
- **全域網格背景**：利用純 CSS 渲染的細緻發光網格，達成無限延伸的空間感。
- **漸層遮罩**：圖表與 3D 物件大量採用青色至透明的漸層，增加科技通透感。

## 🛠️ 第二部分：前端技術堆疊 (Tech Stack)

### 1. Next.js 15 (App Router)
- 採用最新的 App Router 架構，實現檔案系統即路由的開發體驗。
- 全面使用 `"use client"` 優化各類監控圖表的客戶端即時渲染。

### 2. Tailwind CSS v4 (新世代樣式引擎)
- 採用 `@import "tailwindcss";` 配置。
- 自定義 Cyberpunk 佈景主題，強制鎖定暗色模式以維持工業美學一致性。

### 3. Recharts & 3D Digital Twin
- **Recharts**: 實作即時 2D 監控趨勢圖。
- **React Three Fiber (3D)**: 實現 **數據驅動機房**。
  - **U 位精準建模**: 伺服器與機櫃位置根據 `dcim_layout.json` 與 U 位座標動態計算。
  - **動態佈線**: 呈現伺服器與網路齒輪間的即時貝茲曲線連線。

## 🖥️ 第三部分：核心分頁導覽

1. **📊 總覽 (`/`)**: 集中呈現機房健康比例與整體趨勢。
2. **🎛️ 控制 (`/control`)**: 提供工業級撥動開關進行遠端操作模擬。
3. **📈 分析 (`/analysis`)**: 針對單個設備的詳細歷史曲線。
4. **🌐 網路 (`/network`)**: 3D 與 2D 結合的拓撲拓撲顯示。
5. **🔒 工程 (`/engineering`)**: 高權限控制台，需驗證密碼。

## 🔄 第四部分：開發環境啟動 (1150325 規範)

### 啟動流程
1. 進入前端資料夾：`cd frontend`
2. 安裝套件：`npm install`
3. 執行開發伺服器：`npm run dev -- -p 9001`
4. 訪問位址：`http://localhost:9001`

### 常見問題與除錯 (FAQ)
- **Q: 為什麼畫布變白或樣式異常？**
  請刪除 `.next` 資料夾並重啟 `npm run dev`。
- **Q: 如何擴充 3D 內容？**
  請直接修改 `layout/dcim_layout (1).json`，前端會自動感應並渲染新設備，無需手動修改 Three.js 代碼。
