# 🏢 DataCenter 數位孿生智慧監控平台
## 電腦展展覽亮點 & 重點項目簡報

> **產品定位**：面向企業級數據中心的「一站式可視化運維平台」，整合 2D 儀表板、3D 數位孿生與 AI 異常偵測，實現從「看數字」到「看場景」的監控升級。

---

## 一、三大核心亮點 (Key Highlights)

### 🔥 亮點 1：3D 數位孿生機房 — 所見即所得
> **一句話賣點**：不用走進機房，就能「走進」機房。

*   **1:1 虛擬機房**：根據實際機房尺寸建模，機架、伺服器、交換機、CDU 液冷設備一目了然。
*   **即時狀態映射**：每台設備的 LED 指示燈、溫度熱度、CPU 負載都與實體硬體同步。
*   **關機反黑**：設備斷電時，3D 模型立即變為深黑色並熄滅指示燈 — 直覺判斷設備狀態。
*   **自由視角**：支援滑鼠旋轉、縮放、平移，如同操作 Google Earth。

### 🔥 亮點 2：CDU 液冷深度監控 — 看透冷卻系統
> **一句話賣點**：業界首創 CDU 液冷 X-Ray 透視 + 抬頭顯示。

*   **X-Ray 透視模式**：滑鼠懸停即可「看穿」CDU 外殼，觀察內部泵浦運轉狀態。
*   **9 大液冷指標即時顯示**：入水溫、出水溫、流速、壓力、泵浦轉速 (雙泵)、儲液量、閥門開度、洩漏偵測。
*   **流動粒子動畫**：管路中的冷卻液流動速度與實際流速同步，流速異常一眼看出。

### 🔥 亮點 3：模擬 ↔ 真實一鍵切換 — 展場最佳利器
> **一句話賣點**：沒帶伺服器來？沒關係，我們有模擬器。

*   **模擬模式 (Simulation)**：自動產生 18 台伺服器 + 3 台交換機的動態數據，含溫度飆升、CPU 異常等情境。
*   **真實模式 (Real)**：接上一台筆電執行 Agent，即可在數十台「反黑」的模擬設備中，唯獨點亮那一台真實設備 — **對比效果極佳**。
*   **切換無縫**：點擊一個按鈕，2 秒內全場景同步更新。

## 三、技術規格速覽 (Tech Spec)

| 項目 | 規格 |
|:---|:---|
| 前端框架 | Next.js 15 + React Three Fiber |
| 3D 引擎 | Three.js (WebGL) |
| 後端框架 | FastAPI (Python) |
| 訊息佇列 | Apache Kafka |
| 時序資料庫 | InfluxDB |
| 告警資料庫 | MongoDB |
| 通訊協定 | SSE (Server-Sent Events) / REST API |
| 硬體接口 | Redfish / IPMI (預留) / Modbus TCP (CDU) |
| 資料更新頻率 | ≤ 2 秒 |
| 支援設備數 | 已驗證 40+ 節點即時監控 |
