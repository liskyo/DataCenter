---

## 2. 與現況對齊後的摘要

- 前端：`Next.js 16.2.0` + `React Three Fiber` + `Tailwind CSS v4` + `Zustand`
- 後端：`FastAPI` + `Kafka` + `InfluxDB` + `MongoDB` + `Redis`
- 正式主資料流：`HTTP /ingest -> Kafka -> Backend processing -> InfluxDB / MongoDB -> SSE -> Frontend`
- Redfish / IPMI / Modbus 等仍需區分「已落地」與「預留整合」
